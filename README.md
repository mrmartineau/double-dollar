# double-dollar

> `forEach` over DOM elements more easily

## Install from npm

```sh
npm install double-dollar --save
```

## Usage:

```js
var $$ = require('double-dollar');
// or
window.$$ = require('double-dollar');

$$('.foo').forEach(function(item) {
	// do something
});
```
